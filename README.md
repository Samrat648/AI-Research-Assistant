# 📚 AI Research Assistant

A lightweight AI-powered Research Assistant built with Flask and Ollama (Gemma 3).

## Features

- Research Summary
- Key Point Extraction
- Explain Concepts
- Abstract Generator
- Local AI Processing

## Installation

```bash
pip install -r requirements.txt
python app.py
```

Open

```
http://127.0.0.1:5000
```

## Tech Stack

- Python
- Flask
- Ollama
- Gemma 3

> This project is designed to assist with reading and understanding research papers. Users should verify important information with the original sources.
